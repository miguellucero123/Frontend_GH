## [1.0.0] - 2024

### Agregado
- Sistema de login con autenticación por roles
- Panel de usuario para Trabajador/Cliente
- Panel de administración para Jefe
- Gestor documental con estructura de carpetas
- Sistema de chat en tiempo real (WebSocket)
- Vista previa de archivos (PDF, imágenes)
- Sistema de permisos granulares
- Diseño responsivo para móviles, tablets y desktop
- Gestión de proyectos con metadatos completos
- Control de costos (inicial, adicionales, extras, final)
- Aprobación/rechazo de usuarios
- Bandeja de entrada unificada para mensajes
- Sistema de notificaciones
- Utilidades generales (formateo de fechas, monedas, archivos)
- Archivo de configuración centralizado

### Características Técnicas
- JavaScript vanilla (ES6+)
- CSS3 con variables CSS y Grid/Flexbox
- Arquitectura modular
- Manejo de errores robusto
- Validación de formularios
- LocalStorage para persistencia de sesión
- API RESTful client
- WebSocket para comunicación en tiempo real
